# Euphoria_Jewellery_Website

## About
Euphoria Jewellery is a modern e-commerce website designed to showcase and sell elegant jewelry collections online. The platform provides users with a seamless shopping experience through product browsing, category-based navigation, detailed product displays, and a responsive user interface.

## Features
-💍 Product catalog with category-based filtering
-🔍 Search functionality for quick product discovery
-🛒 Shopping cart and checkout system
-📱 Fully responsive design for mobile and desktop users

## Technologies Used
HTML | CSS | JavaScript | PHP | MySQL

## How to Run
1. Clone this repo
2. Import db.sql in phpMyAdmin
3. Open index.html in browser
